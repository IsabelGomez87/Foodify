# Foodify

How tu run this app:

1- Download the project

2- Open project in your code editor

3- Run next command for install dependencies
```bash
npm install
```

4- Open server folder and run the command
```bash
npm start
```

5- Open frontend folder and run the command 
```bash
npm start
```
